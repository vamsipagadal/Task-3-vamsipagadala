# Project 3 – Architectural Shell & Interior Layout

## Project Overview

Project 3 is a comprehensive AutoCAD project consisting of two interconnected drawings: an Architectural Shell and an Interior Layout. The Architectural Shell defines the building's structural framework, including walls, doors, windows, and room divisions. The Interior Layout complements the shell by illustrating furniture placement, interior planning, and functional space utilization. Together, these drawings demonstrate the complete design and documentation workflow for a residential building.

## Project Files

* **ARCH_SHELL(3).dwg** – Architectural shell containing the structural floor plan with walls, doors, windows, and room layout.
* **INTERIOR_LAYOUT_1(4).dwg** – Interior layout containing furniture placement, interior fixtures, and space planning details.

## Objectives

* Create a precise architectural floor plan.
* Design a functional and aesthetically organized interior layout.
* Apply standard AutoCAD drafting practices.
* Maintain accurate dimensions and annotations.
* Organize drawings using layers and reusable blocks.
